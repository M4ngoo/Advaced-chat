Αρχικά, ανοίγουμε 3 φορές java Server με διαφορετική εισαγωγή στο select server κάθε φορά (την 1η φορά με 0 , την 2η με 1 , την 3 με 2). Έπειτα μπορούμε να ανοίξουμε όσα
clients θέλουμε (java Client). Πληκτρολογώντας \help φαίνονται όλες οι επιλογές του Client. Το /downloadfile δουλεύει διαφορετικά στο topic απ΄οτι στο /showposts. Στο topic
πρέπει δίπλα να εισάγεις το μοναδικό id του file (πχ /downloadfile 35) ενώ στο /showposts πρέπει να εισάγεις το όνομα του αρχείου (πχ /downloadfile video.mp4).
